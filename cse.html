B.Tech in Computer Science and Engineering (CSE)

1. Course Description

The B.Tech in Computer Science and Engineering program provides students with a strong foundation in computer systems, software development, algorithms, and data structures. The course integrates theory and practical skills for building a career in computing and technology fields.

2. Duration

This is a 4-year full-time undergraduate program divided into 8 semesters.

3. Subjects Covered

  • Programming in C & C++
  • Data Structures & Algorithms
  • Operating Systems
  • Database Management Systems
  • Computer Networks
  • Web Technologies
  • Machine Learning Basics
  • Cloud Computing

4. Faculty Involved

The department is staffed by highly qualified faculty including professors, associate professors, and assistant professors specializing in various domains such as software engineering, AI, and system architecture.

5. Career Opportunities

  • Software Developer
  • Systems Engineer
  • Data Analyst
  • Web Developer
  • Cloud Solutions Architect
  • Machine Learning Engineer
  • Cybersecurity Analyst
